# Frontend UI Development

## Instructions

1. **Page & Layout Structure**

   - Define responsive page layouts  
   - Use semantic HTML  
   - Implement grid or flexbox systems  

2. **Component Development**

   - Build reusable UI components  
   - Manage component state and props  
   - Ensure component reusability and consistency  

3. **Styling & Responsiveness**

   - Apply modern CSS or utility frameworks  
   - Maintain consistent spacing and typography  
   - Ensure mobile, tablet, and desktop support  

## Best Practices

- Follow component-based architecture  
- Keep styles modular and reusable  
- Use mobile-first design  
- Maintain design consistency  
- Optimize for performance and accessibility  

## Example Structure

```html
<section class="page">
  <header class="navbar">
    <h1 class="logo">My App</h1>
  </header>

  <main class="content">
    <div class="card">
      <h2>Component Title</h2>
      <p>Reusable component content</p>
      <button class="primary-btn">Action</button>
    </div>
  </main>
</section>
